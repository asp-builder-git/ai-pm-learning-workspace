# Session Notes

Raw debriefs from PM pipeline sessions using the agent system. Each note captures:

- **What was tried** — the experiment or build
- **What went wrong** — failures, root causes, wasted iterations
- **The lesson** — how to avoid it next time
- **Takeaway** — actionable next step

These aren't polished documentation. They're post-mortems written immediately after a session, while the friction is fresh. Come here when you feel a pattern repeating — the answer is probably in an old note.
