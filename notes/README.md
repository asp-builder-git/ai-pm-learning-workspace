# Daily Learnings

Raw notes from each session. May or may not relate to the PM agent pipeline — this is just a log of what I learned today.

Each note captures:

- **What was tried** — the experiment or build
- **What went wrong** — failures, root causes, wasted iterations
- **The lesson** — how to avoid it next time
- **Takeaway** — actionable next step

These aren't polished documentation. They're debriefs written while the friction is fresh. Come here when you feel a pattern repeating — the answer is probably in an old note.
